# Jack-in and Connect

Demos: (disconnect after each)
* Jack in to project less show hello.clj
* Jack in to pirate-lang using deps.edn
* Copy the jack-in command line, and connect.
* Jack-in using pirate-lang sequence
* cd projects/pirate-lan, bb docker-repl, connect
* jack-in to docker-repl, custom command line

When to use which?

Jack-in:
* Convenience
* One less thing to learn before using the REPL
* Managed process

Connect:
* REPL started by your infrastructure
* Remote REPL (container, server)
* REPL started by another tool
* Not all REPLs can be started by Jack-in
