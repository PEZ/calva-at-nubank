# Joyride

<div class="col">

**Script VS Code with ClojureScript** - SCI-powered, live REPL

- Full VS Code API + all extension APIs
- User scripts & workspace scripts, activation scripts
- Script paths or code in keyboard shortcuts
- Full Interactive Programming
- Copilot Tools (no MCP (but Backseat Driver))

#### Flares

- WebView panels & sidebar views from Hiccup/HTML
- Embed Scittle apps (Scittle has a REPL too)

#### Examples

[live_examples.cljs](../.joyride/src/live_examples.cljs), [flares_examples.cljs](../.joyride/src/flares_examples.cljs), [ttt_flare.cljs](../.joyride/src/scittle-replicant-tic-tac-toe/ttt_flare.cljs)

- next-slide, showtime, URL flares manager, Keybindings Palette
- Pastedown
- Git Fuzzy Search
- JS-REPL 😀 [scrap.js](../etc/scrap.js)
