# Copilot/AI Learnings

<div class="col">

### Mindset

- **Config, scripts, harness > instructions**
- **Never let a mistake go to waste**
  - Teach the agent to learn (watch out what it learns!)
- **Spend on the plans** - Execution demands less brains

### My Environment is full of Pitfalls!

- AI forces you to **fix configuration glitches and gaps**
  - Linting — keep the lint report clean (a trust issue)
  - Flaky tests, noisy tool output, Docker not starting
  - Always start watchers, always connect your REPLs

</div>
<div class="col">

### AGENTS.md / copilot-instructions.md

- Be **super picky** with what goes in
- Don't auto-generate — curate
- Common mistakes are candidates, but fixing the pitfall > documenting it
- As models get smarter, discovery beats documentation

### Prompting & Workflow

- Custom Prompts
- Custom agents / custom instructions
- Skills
- **Subagents** - Subagent everything
- **Parallel subagents** — parallelize everything
- While tooling is building: go heavy with agents analyzing, planning, reviewing
  - Then relax gradually
- `#askQuestions` — let the agent ask you
