# Custom REPL Commands

## Demo plan (pirate-lang project)

* Show a simple snippet: Test Current Var (`ctrl+alt+space t`)
* Run from custom commands palette (`ctrl+alt+space space`)
* **Malli schema inference** (`ctrl+alt+m ctrl+alt+enter`) - select sample data, infer schema
* **RCF inline tests** (`ctrl+alt+c ctrl+alt+enter`) - run `(tests ...)` blocks in pirate_lang.clj
* Mention Snippets Inside Deps - library authors can ship snippets
